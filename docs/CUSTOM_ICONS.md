# カスタムアイコンの設定方法

## 📁 アイコンファイルの配置場所

`assets/` ディレクトリに以下のファイルを配置してください：

- `user_icon.png` - ユーザー（質問者）のアイコン
- `assistant_icon.png` - アシスタント（回答者）のアイコン

## 🎨 アイコンの推奨仕様

- **形式**: PNG（透明背景推奨）
- **サイズ**: 64x64px 〜 128x128px が最適
- **背景**: 透明（.png）または白背景
- **スタイル**: 丸型または角丸のアイコンが美しく表示されます

## 📝 設定手順

1. アイコン画像を準備
   - ユーザー用: `assets/user_icon.png`
   - アシスタント用: `assets/assistant_icon.png`

2. ファイルを配置
   ```
   assets/
   ├── concierge_logo.png
   ├── user_icon.png      ← 新規追加
   └── assistant_icon.png ← 新規追加
   ```

3. アプリを再起動
   - Streamlitが自動的にアイコンを検出して使用します

## 💡 アイコンが見つからない場合

アイコンファイルがない場合は、デフォルトのStreamlitアイコンが使用されます。

## 🎯 アイコンの例

- **ユーザー**: 人のシルエット、ユーザーアイコン、質問マークなど
- **アシスタント**: ロボット、コンシェルジュキャラクター、案内人のアイコンなど
